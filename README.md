# Stockity / Focus Vision AI — готовий виправлений код

файли index.html, app.js, styles.css, manifest.json можуть лежати прямо в корені репозиторію.

## Railway Variables

Додай у Railway → Variables:

OPENAI_API_KEY=твій_openai_api_key
BOT_TOKEN=токен_бота_з_BotFather
WEBAPP_URL=https://твій-домен.up.railway.app
APP_NAME=Stockity AI

## Railway Start Command

python main.py
